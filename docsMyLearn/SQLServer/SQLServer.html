SQL Server Document

tip
SQL Server 是由微软（Microsoft）开发的关系型数据库管理系统（RDBMS），专门用于存储、管理和检索数据。SQL Server 采用结构化查询语言（SQL）作为主要的数据库查询语言，并为企业和应用程序提供数据管理解决方案
note

数据查询语言 (DQL)：用于查询数据，例如 SELECT 语句
数据定义语言 (DDL)：用于定义和修改数据库结构，例如 CREATE、ALTER 和 DROP 语句
数据操作语言 (DML)：用于操作数据库中的数据，例如 INSERT、UPDATE 和 DELETE 语句
数据控制语言 (DCL)：用于控制访问权限，例如 GRANT 和 REVOKE 语句
事务控制语言 (TCL)：用于管理事务，例如 COMMIT 和 ROLLBACK 语句

CURD​

Create（创建）、Update（更新）、Read（读取）、Delete（删除）

数据类型​

sql_variant (Transact-SQL)

SELECT: 从数据库中查询数据​

  • SELECT column1, column2 FROM table_name WHERE condition

INSERT INTO：向表中插入数据​

  • INSERT INTO table_name (column1, column2) VALUES (value1, value2);

UPDATE：更新表中的数据​

  • UPDATE table_name SET column1 = value1 WHERE condition

DELETE：从表中删除数据​

  • DELETE FROM table_name WHERE condition

CREATE TABLE：创建新表​

  • CREATE TABLE table_name ( column1 datatype, column2 datatype, ... )

ALTER TABLE：修改表结构​

  • ALTER TABLE table_name ADD column_name datatype;

DROP TABLE：删除表​

  • DROP TABLE table_name
