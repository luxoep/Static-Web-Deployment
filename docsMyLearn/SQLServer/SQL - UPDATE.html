Update

向表中插入数据​

基本更新​

  • 更新表中所有行的指定列
    • UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition

使用 JOIN 更新​

  • 基于两个表的关联更新记录
    • UPDATE a
      SET a.column_name = b.column_name
      FROM table1 a
      INNER JOIN table2 b ON a.common_column = b.common_column

使用子查询更新​

  • 使用子查询的结果来更新记录
    • UPDATE table_name
      SET column_name = (SELECT expression FROM another_table WHERE condition)
      WHERE condition

更新多个列​

  • 同时更新多个列的值
    • UPDATE table_name
      SET column1 = value1, column2 = value2, column3 = value3
      WHERE condition

更新所有行​

  • 在没有 WHERE 子句的情况下，更新表中的所有行（谨慎使用）
    • UPDATE table_name SET column_name = value

使用 CASE 语句更新​

  • 根据不同的条件更新不同的值
    • UPDATE table_name
      SET column_name = CASE
      WHEN condition1 THEN value1
      WHEN condition2 THEN value2
      ELSE value3
      END
      WHERE condition

使用 TOP 子句更新​

  • 更新表中的前几行记录
    • UPDATE TOP (number) table_name
      SET column_name = value
      WHERE condition

使用 OUTPUT 子句​

  • 查看更新操作前后的数据
    • UPDATE table_name
      SET column_name = value
      OUTPUT deleted.column_name, inserted.column_name
      WHERE condition

使用 MERGE 语句​

  • 结合了 INSERT、UPDATE 和 DELETE 操作的复杂数据修改
    • MERGE target_table AS target
      USING source_table AS source
      ON target.common_column = source.common_column
      WHEN MATCHED THEN
      UPDATE SET target.column1 = source.column1
      WHEN NOT MATCHED BY TARGET THEN
      INSERT (column1, column2)
      VALUES (source.column1, source.column2)
