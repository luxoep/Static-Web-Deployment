Select

从数据库中查询数据​

字符串函数​

  • 连接两个或多个字符串
    • SELECT 'Hello'+ ' '+ 'World' AS Greeting
      • --> Hello World
  • SUBSTRING()：从字符串中提取子字符串
    • SELECT SUBSTRING('Hello World', 1, 5) AS SubStr
      • --> Hello
  • LEN()：返回字符串的长度。
    • SELECT LEN('Hello') AS StringLength
      • --> 5
  • UPPER() 和 LOWER()：将字符串转换为大写或小写。
    • SELECT UPPER('hello') AS UpperCase
      • --> HELLO
    • SELECT LOWER('HELLO') AS LowerCase
      • --> hello
  • REPLACE()：在字符串中替换子字符串。
    • SELECT REPLACE('Hello World', 'World', 'SQL') AS NewString
      • --> Hello SQL

日期和时间函数​

  • GETDATE()：返回当前日期和时间
    • SELECT GETDATE() AS CurrentDateTime
      • --2024-09-18 12:34:56.789 (具体取决于当前时间)
  • DATEADD()：向日期添加指定的时间间隔
    • SELECT DATEADD(DAY, 10, '2024-09-18') AS NewDate
      • --> 2024-09-28
  • DATEDIFF()：返回两个日期之间的差异
    • SELECT DATEDI(i)FF(DAY, '2024-09-01', '2024-09-18') AS DateDifference
      • --> 17
  • CONVERT()：将日期时间格式化为指定的格式
    • SELECT CONVERT(VARCHAR(10), GETDATE(), 120) AS FormattedDate
    • 101：mm/dd/yyyy
    • 103：dd/mm/yyyy
    • 112：yyyymmdd
    • 120：yyyy-mm-dd
    • ....
      • --> 2024-09-18
  • YEAR()、MONTH()、DAY()：提取日期的年份、月份或天数
    • SELECT YEAR(GETDATE()) AS CurrentYear, MONTH(GETDATE()) AS CurrentMonth, DAY(GETDATE()) AS CurrentDay
      • --> 2024, 9, 18

聚合函数​

  • COUNT() 返回行数或某列的非空值数量
  • SUM() 返回某列数值的总和
  • AVG() 返回某列数值的平均值
  • MAX() 和 MIN() 分别返回某列的最大值和最小值

条件与转换函数​

  • ISNULL()：替换 NULL 值为指定值
    • SELECT ISNULL(NULL, 'Default Value') AS ReplacedValue --> Default Value
  • COALESCE()：返回第一个非 NULL 值
    • SELECT COALESCE(NULL, NULL, 'First Non-NULL Value') AS NonNullValue --> First Non-NULL Value
  • CAST() 和 CONVERT()：用于数据类型转换
    • SELECT CAST(123.45 AS INT) AS IntValue --> 123
    • SELECT CONVERT(VARCHAR(10), GETDATE(), 103) AS FormattedDate --> 18/09/2024
  • CASE：类似于 IF-ELSE 的条件表达式
    • SELECT ID, USENAME, LOGINID, (CASE WHEN [GROUP] = 2 THEN '1' ELSE '0' END) AS NEWGROUP FROM CLASS

嵌套查询（子查询）​

  • 查询员工ROLE大于ROLE平均数的员工
    • SELECT * FROM CLASS AS CLASS_NEW WHERE (ROLE > (SELECT AVG(ROLE) AS ROLEAVG FROM CLASS AS CLASS_old))

GROUP BY 查询​

  • GROUP BY 用于将数据按照某一列进行分组，并且通常与聚合函数（如 COUNT()、SUM()、AVG() 等）结合使用
  • 统计每个ROLE的LOGINID总数和平均ID
    • SELECT ROLE, COUNT(LOGINID) AS LOGINIDCOUNT, AVG(ID) AS AVERAGEID FROM CLASS GROUP BY ROLE

ORDER BY 查询​

  • ORDER BY 用于根据一列或多列对查询结果进行排序。默认是升序 (ASC)，可以显式指定降序 (DESC)
  • 查询员工信息，按GROUP降序排列，如果工资相同则按ROLE升序排列
    • SELECT * FROM CLASS ORDER BY [GROUP], ROLE DESC

多表联合查询（JOIN 查询）​

  • 常见的联合查询类型有 INNER JOIN、LEFT JOIN、RIGHT JOIN 和 FULL JOIN
  • INNER JOIN：只返回两个表中都匹配的记录
  • LEFT JOIN：返回左表中的所有记录，即使右表中没有匹配项
  • RIGHT JOIN：返回右表中的所有记录，即使左表中没有匹配项

SELECT CLASS.USENAME, CLASS.[GROUP], [GROUP].GROUPNAME FROM CLASS INNER/LEFT/RIGHT JOIN [GROUP] ON CLASS.[GROUP] = [GROUP].GROUPID
